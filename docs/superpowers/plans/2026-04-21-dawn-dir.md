# `.dawn` Directory Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Move `dawn.generated.d.ts` from the user's source tree (`src/app/`) into a `.dawn/` directory at the project root.

**Architecture:** `findDawnApp()` gains a `dawnDir` property computed as `join(appRoot, ".dawn")`. All consumers that write generated types (`typegen`, vite plugin) use `app.dawnDir` instead of `app.routesDir`. The starter template and gitignore are updated to match.

**Tech Stack:** TypeScript, Node.js, Vitest
