# Support

## Questions

Use GitHub Discussions for usage questions, design discussions, and integration help:

https://github.com/cacheplane/dawnai/discussions

The public documentation is available at https://dawn-ai.org/docs/getting-started.

## Bugs

Use GitHub Issues for reproducible bugs. Include the smallest reproduction you can provide, package versions, Node.js version, pnpm version, and the command or route involved.

## Security

Do not file security issues publicly. Follow [SECURITY.md](./SECURITY.md).
